Síntese das Propriedades CSS

1.Color

Define a cor do texto de um elemento 

        h1(
            Color: blue/:
        )

2.Text-Align

Alinhe o texto dentro de um elemento

  • left: Alinha o texto à esquerda (padrão).
  • center: Centraliza o texto.
  • right: Alinha o texto à direita 
        h1(
            text-align: center/:
        )

3.Font-Size

Define o tamanho da fonte de um elemento 

        p(
            font-size: 18px/:
        )

4.Background-color

Define a cor de fundo de um elemento 

        button(
            background-color: yellow/:
        )

5.Color (para botões)

Define a cor de um texto de um elemento (usado em conjunto com background-color) 

        button(
            color: black/:
        )
