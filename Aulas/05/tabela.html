Aula Tabelas

Introdução às tabelas em HTML

Tabelas são estruturas em HTML que permitem a organização de dados em linhas e colunas. São comumente usadas para exibir calendários, horários, listas de prços, etc.

Estrutura básica de uma Tabela

Estrutura básica de uma Tabela em HTML é composta por várias tags:

  • table: Define o inicio e fim de uma tabela.
  • tr: Define uma linha na tabela
  • td; Define uma célula dentro de uma linha (dados)
  • th: Define uma céula de cabeçalho

Exemplo 1: tabela básica

Nome Idade Profissão
Ana 28 Engenheira
Bruno 34 Médico

Explicação:

  • Nesse exemplo, usamos th para o cabeçalho e td para os dados.
  • O atributo border="1" é usado para adicionar à tabela.

Mesclagem de células

A mesclagem pode ser feita tanto em colunas quanto em linhas.

  • Colspan: Mescla células horizontalmente (Colunas)
  • Rowspan: Mescla células verticalmente (linhas)
Nome Idade profissão
Carlos 30 Professor
31 Arquiteto
Daniela Designer Gráfica

Explicação:

  • rowspan="2" a célula "Carlos" se expande verticalmente para cobrir duas linhas
  • colspan="2" A célula "Designer Gráfico" se expande horizontalmente para cobrir duas colunas.

Adicionando um cabeçalho e um rodapé à tabela

Além das linhas de dados, uma tabela pode ter um cabeçalho (thead), um corpo (body) e um (tfoot)

Produto Quantidade Preço Unitário
Caneta 10 R$ 2,00
Caderno 5 R$ 15,00
Total: R$ 95,00

Explicação:

  • thead Define o cabeçalho da tabela
  • tbody Define o corpo da tabela, onde fica os dados
  • tfoot Define o rodapé, ideal para somatórias ou resumos
