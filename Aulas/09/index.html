Introdução à Tag <img>

A tag <img> em HTML é usada para incorporar imagens em uma página web. Ela é um elemento vazio. ou seja, não possui uma tag de fechamento e contém apenas atributos que definem o conteúdo visual.

Atributos princípais:

  • Src (Source): Define o caminho da imagem que será exibida. Poder ser uma URL externa ou um caminho dentro do servidor.

    Exemplo 1:

    <img src="./image.png" style="width: 100%;">

  • alt (alternative text): fornece um texto alternativo para ser exibido caso a imagem não possa ser carregada. Ele também melhora a acessibilidade para leitores de tela.

    <img src="0f9f470ad8025b229719a415f03bd3a0763ed602_00.gif" style="width: 100%;">

Tipos comuns de imagens

Os formatos de imagens mais utilizados na web são:

  • (.jpg ou .jpeg): Formato utilizado amplamente para fotografias e imagens com muitos detalhes e cores. Possui compressão com perda de qualidade (lossy).
  • (.png): Usado para imagens com fundo transparente ou que necessitam de alta qualidade, sem perda de detalhes. Utiliza compreensão sem perda (lossless).
  • GIF (.gif) Formato ideal para imagens animadas e pequenos ícones. Suporta transparência, mas tem um número limitdo de cores.
